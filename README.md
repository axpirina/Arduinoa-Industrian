## Aurretiko jakintzak

Ez da bereziki elektronika edo programazio aurrekarririk behar, baina ongi etortzen da C edo antzeko hizkuntzen jakintza pixkat. 

```bash
digitalWrite(13, HIGH); // sets the digital pin 13 on
```
